🌿 About the Project

This system uses a moisture sensor to detect soil dryness and activates a pump accordingly. It's designed for farmers and gardeners to automate watering, save water, and increase productivity.

🔧 Components Used

Moisture Sensor

Detects soil dryness.

Arduino UNO

Main controller.

Relay Module

Controls the pump.

Water Pump

Waters the plants.

Power Supply

Provides energy.

⚙️ How It Works

1️⃣ Moisture sensor checks soil.
2️⃣ Arduino reads the data.
3️⃣ If dry, signal sent to relay.
4️⃣ Pump gets activated.
5️⃣ System stops when soil is wet.
